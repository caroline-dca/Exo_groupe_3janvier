<select>

Element de formulaire : c'est un menu déroulant liste.
Les balises <select> </select> incluent forcément plusieurs balises <option> </option>.
Celles-ci déterminent les divers éléments de la liste : le texte inclus dans les balises est ce que l'utilisateur lit ; l'attribut value permet le traitement de l'information.
La valeur de l'élément sélectionné par l'utilisateur sera "true" ou 1 (vrai), tandis que le reste sera "false" ou 0 (faux).
