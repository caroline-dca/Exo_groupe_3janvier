<Title>

La balise <title> est obligatoire dans tous les documents HTML et définit le titre du document.

L'élément <title>:

  • définit un titre dans la barre d'outils du navigateur
  • fournit un titre pour la page lorsqu'elle est ajoutée aux favoris
  • affiche un titre pour la page dans les résultats du moteur de recherche

Cet élément ne peut contenir que du texte, les balises qu'il contiendrait seraient ignorées.

Les deux balises sont nécessaires. Si </title> est absent, le navigateur peut ignorer le reste de la page.
