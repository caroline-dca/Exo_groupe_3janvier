id=

id n'est pas une balise, mais un attribut. Il permet de donner un identifiant unique à une balise, ce qui sera utile pour d'autres parties de code (si l'on veut le manipuler grâce à un script, le mettre en forme avec une feuille de style, le cibler pour un lien...).

I. Historique
